MODULE 1 – Diving into C: Basics

Theory Points:

Introduction to C: History, features, compilation process.

Variables, constants, and data types (int, float, double, char, void).

Operators: Arithmetic, relational, logical, bitwise, assignment.

Input/Output: scanf(), printf().

Control structures: if, if-else, switch-case, loops (for, while, do-while).

Comments and code documentation.

Tables:

C Data Types: Size, Range, Description
| Type | Size (bytes) | Range | Use |
|------|-------------|-------|-----|
| int | 4 | -2,147,483,648 to 2,147,483,647 | Integer numbers |
| char | 1 | -128 to 127 or 0 to 255 | Characters |
| float| 4 | 3.4e-38 to 3.4e+38 | Floating-point numbers |
| double| 8 | 1.7e-308 to 1.7e+308 | High precision numbers |

Minimal Code Examples:

#include <stdio.h>
int main() {
    int a = 10;
    float b = 3.5;
    printf("Integer: %d, Float: %.2f\n", a, b);
    return 0;
}

#include <stdio.h>
int main() {
    int i;
    for(i=1;i<=5;i++)
        printf("%d ", i);
    return 0;
}


Learning Outcomes:

Understand C syntax, data types, and operators.

Write simple programs with conditional statements and loops.

MODULE 2 – Beyond C: Advanced Concepts & Practical Applications

Theory Points:

Functions: Declaration, definition, calling, return types, parameters.

Recursion: Factorial, Fibonacci, and recursive algorithms.

Arrays: 1D, 2D arrays, operations, initialization.

Strings: Declaration, input/output, string functions (strlen, strcpy, strcmp).

Structures: Custom data types to group variables.

Scope and lifetime: Local, global, static variables.

Minimal Code Examples:
Recursion:

#include <stdio.h>
int factorial(int n) {
    if(n <= 1) return 1;
    return n * factorial(n-1);
}
int main() {
    printf("Factorial of 5: %d\n", factorial(5));
    return 0;
}


Array Example:

#include <stdio.h>
int main() {
    int arr[5] = {1,2,3,4,5};
    for(int i=0;i<5;i++) printf("%d ", arr[i]);
    return 0;
}


Structure Example:

#include <stdio.h>
struct Student {
    int id;
    char name[50];
    float marks;
};
int main() {
    struct Student s1 = {1, "Alice", 95.5};
    printf("ID:%d Name:%s Marks:%.2f\n", s1.id, s1.name, s1.marks);
    return 0;
}


Learning Outcomes:

Use functions, arrays, and structures in C programs.

Solve problems using recursion and modular programming.

MODULE 3 – Data Structures, Pointers, and File Systems

Theory Points:

Pointers: Memory addresses, dereferencing, pointer arithmetic, pointer to pointer.

Dynamic memory: malloc(), calloc(), realloc(), free().

Data Structures: Arrays, linked lists, stacks, queues, and their operations.

File Handling: Reading and writing text and binary files, error handling.

Minimal Code Examples:
Pointer Example:

#include <stdio.h>
int main() {
    int a = 10;
    int *ptr = &a;
    printf("Value: %d, Address: %p\n", *ptr, ptr);
    return 0;
}


Dynamic Memory:

#include <stdio.h>
#include <stdlib.h>
int main() {
    int *arr = (int*) malloc(5 * sizeof(int));
    for(int i=0;i<5;i++) arr[i] = i+1;
    for(int i=0;i<5;i++) printf("%d ", arr[i]);
    free(arr);
    return 0;
}


File I/O Example:

#include <stdio.h>
int main() {
    FILE *fp = fopen("data.txt","w");
    if(fp!=NULL) { fprintf(fp,"Hello C!\n"); fclose(fp); }
    return 0;
}

MODULE 6 – Debugging C Code
